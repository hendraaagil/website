An education platform based in Malaysia, designed to help SPM students improve their academic performance through structured, engaging learning experiences.

I serve as the primary frontend engineer and am responsible for maintaining the platform end-to-end — from feature development and UI implementation to performance and deployment.

### Tech Stack

- **Framework**: Next.js
- **State Management**: Redux Toolkit, TanStack Query
- **Database & Auth**: Prisma, Supabase
- **Styling & Animation**: Tailwind CSS, GSAP
- **Forms & Validation**: React Hook Form + Zod
- **i18n**: next-intl
